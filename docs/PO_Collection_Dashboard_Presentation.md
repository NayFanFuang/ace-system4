# PO Collection Tracking Dashboard
### ระบบติดตามการเก็บเงิน PO ระหว่างบัญชี ↔ โปรเจกต์
**ACE System · AirConnect Engineering**
เสนอวันที่ 14 มิ.ย. 2026

---

## 1. ปัญหา (Why)

ทุกวันนี้คำถามสำคัญของผู้บริหารตอบยาก เพราะข้อมูลกระจายหลายหน้า:

> *"PO ตัวไหนเก็บเงินจาก Huawei แล้ว? ตัวไหนยัง? ตัวไหนโดน Reject?
> เงินค้างเก็บอยู่เท่าไหร่ และตอนนี้ค้างอยู่ที่ **บัญชี** หรือ **โปรเจกต์**?"*

- ข้อมูลการเก็บเงิน (บิล Huawei) และการจ่ายเงิน (ค่าแรง DTE) แยกกันคนละหน้า
- ไม่มีภาพรวมว่างานค้างอยู่ขั้นไหน ใครต้องเป็นคน action ต่อ
- ไม่มีตัวชี้วัดว่า PO ตัวไหน "ดอง" นานผิดปกติ → เก็บเงินช้า กระทบกระแสเงินสด

---

## 2. ทางออก (What)

**Dashboard หน้าเดียวจบ** ที่รวมเงินทั้ง 2 ขาของ PO ไว้ในที่เดียว
และตอบคำถามข้างต้นได้ใน **5 วินาที**

| ขาเงิน | คำถามที่ตอบ |
|---|---|
| 💰 **เก็บเงินเข้า** (Bill Huawei) | เก็บแล้ว / เก็บบางส่วน (AC1) / ยังไม่เก็บ / Reject |
| 💸 **จ่ายเงินออก** (จ่าย DTE) | จ่ายแล้ว / ค้างจ่าย |

> เข้าใช้งานที่เมนู **Finance → ติดตามเก็บเงิน PO** (`/finance/po-tracking`)

---

## 3. ความสามารถหลัก (Features)

### 🎯 การ์ดตัวชี้วัด (KPI) 8 ตัว
มูลค่า PO รวม · เก็บเงินแล้ว (+ % collection rate) · ค้างเก็บ · ถูก Reject ·
จ่าย DTE แล้ว · ค้างจ่าย DTE · เก็บบางส่วน · **PO ค้างนานเกิน 14 วัน**

### 📊 ภาพรวม (Overview)
- **สัดส่วนการเก็บเงิน** — แถบสีแสดงมูลค่าที่เก็บแล้ว/บางส่วน/ยังไม่เก็บ/reject
- **Pipeline** — PO แต่ละตัวอยู่ขั้นไหน (รีวิวบัญชี → วางแผนโปรเจกต์ → อนุมัติ → หน้างาน → ปิดบิล)
- **ค้างอยู่ที่ใคร** — แยกชัดว่างานค้างที่ บัญชี หรือ โปรเจกต์ (ความรับผิดชอบ)
- แยกตาม **Vendor** และ **โปรเจกต์**

### 📋 รายการ PO (Detail)
ตารางทุก PO พร้อม: เลขที่ · ไซต์ · มูลค่า · เจ้าของงานปัจจุบัน · สถานะ ·
**ป้ายสถานะเก็บเงิน** · ความคืบหน้า AC1/AC2 · สถานะจ่าย DTE · **จำนวนวันที่ค้าง**
+ ค้นหา และ **Export Excel**

### ⏱️ ค้างนาน (Aging Watchlist) — จุดเด่นมืออาชีพ
รายการ PO ที่ยังเก็บเงินไม่ได้ เรียงตามจำนวนวันที่ค้าง
ไฮไลต์สีแดง (>14 วัน) / เหลือง (>7 วัน) → **เห็นทันทีว่าต้องเร่งตัวไหน ใครดองงาน**

## 4. คุณค่าทางธุรกิจ (Business Value)

| ด้าน | ผลลัพธ์ |
|---|---|
| 🔎 **ความโปร่งใส** | บัญชี-โปรเจกต์-ผู้บริหาร เห็นข้อมูลชุดเดียวกัน real-time |
| ⚡ **เก็บเงินเร็วขึ้น** | Aging แจ้งเตือน PO ดอง → ลดรอบเก็บเงิน, ดีต่อกระแสเงินสด |
| 🎯 **ความรับผิดชอบชัด** | รู้ว่างานค้างที่ใคร ลดการโยนงานไปมา |
| 📤 **พร้อมตรวจสอบ** | Export Excel เป็นหลักฐาน/รายงานผู้บริหารได้ทันที |

## 6. จุดแข็งทางเทคนิค

- ✅ **ต่อยอดจากข้อมูลเดิมทั้งหมด** — ใช้ฟิลด์ที่ระบบบันทึกอยู่แล้ว
  (`hw_billed_at`, AC1/AC2 invoice, `dte_paid_at`, `workflow_status`, `current_owner_role`)
- ✅ **ไม่ต้องแก้โครงสร้างฐานข้อมูล / ไม่ต้อง migrate** — เปิดใช้ได้ทันที
- ✅ **API เดียวจบ** ประสิทธิภาพดี — รวม KPI + breakdown + รายการในครั้งเดียว
- ✅ ทดสอบ build ผ่านทั้ง frontend และ backend แล้ว
