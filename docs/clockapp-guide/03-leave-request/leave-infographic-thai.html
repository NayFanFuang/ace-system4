Leave Request

สำหรับยื่นคำขอลา ตรวจสถิติวันลา ติดตามสถานะคำขอ และดูเงื่อนไขการลาในหน้า Leave
1

กรอกคำขอลา

เลือกประเภทลา วันที่เริ่ม-สิ้นสุด ระบุเหตุผล และตรวจจำนวนวันก่อนกด Submit Request

2

ตรวจสรุปและสถิติ

ดู Request Summary, Leave Statistics และ Process Flow เพื่อเข้าใจลำดับอนุมัติ

3

ติดตามสถานะคำขอ

ตรวจ My Leave Requests, สถานะ Pending / Approved / Rejected และดู Leave Rules

ขั้นตอนใช้งาน Leave

  • Login เข้า ClockApp แล้วเปิดแท็บ Leave
  • เลือก Leave Type ให้ตรงกับเหตุผลการลา
  • เลือก Start Date และ End Date ให้ถูกต้อง
  • กรอก Reason ให้ชัดเจนเพื่อให้หัวหน้าอนุมัติได้เร็ว
  • ตรวจ Total Days แล้วกด Submit Request

หลังยื่นคำขอ

  • ตรวจคำขอใน My Leave Requests
  • ใช้ Refresh เมื่อต้องการอัปเดตสถานะล่าสุด
  • หากคำขอยัง Pending ให้รอหัวหน้างานพิจารณา
  • ตรวจ Leave Statistics เพื่อดูจำนวนวันที่ใช้ไป
  • อ่าน Leave Rules ก่อนยื่นคำขอครั้งถัดไป
