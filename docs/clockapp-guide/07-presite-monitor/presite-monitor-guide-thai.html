จุดที่ DTE ควรเช็ก

เห็นไซต์หรือไม่ถ้าไม่เห็นไซต์ ให้แจ้ง PM ตรวจการ Assign / Plan DTE และสิทธิ์ PER_SITE

Report แนบแล้วหรือยังเมื่ออัปโหลด .rar สำเร็จ ตารางจะแสดง Report Done หรือไฟล์แนบให้ตรวจ
มี Check Fail ไหมเปิดแถวไซต์เพื่ออ่าน notes และดำเนินการแก้ไขก่อนส่งใหม่

เมนูที่ใช้บ่อย

  • Daily: ดูกิจกรรม DT ของวันนี้และการ Sync จาก ClockApp
  • SSV Pre-Site: ดูไซต์ SSV รายไซต์ พร้อม Stage / SLA / Rework
  • PAC Pre-Site: ดูงาน Cluster / SSOA และรอบงานที่เกี่ยวข้อง
  • DTE Details: ดูข้อมูล DTE, workload และกิจกรรมล่าสุด

แนวทางแก้ปัญหาเร็ว

  • กด Refresh หลัง Clock หรือ Upload Report ถ้าสถานะยังไม่เปลี่ยน
  • ถ้าไซต์ไม่ขึ้นใน ClockApp หรือ Monitor ให้แจ้ง PM พร้อมชื่อไซต์
  • ถ้า Report ไม่ขึ้น ให้ตรวจว่าอัปโหลดผ่านหน้า Report Upload แล้ว
  • ถ้า Check Fail ให้เปิดรายละเอียดไซต์ อ่าน notes แล้วแก้ตามรายการ
