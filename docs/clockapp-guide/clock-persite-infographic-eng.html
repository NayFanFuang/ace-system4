ACE ClockApp User Guide

Clock In / Clock Out PER-SITE

For DTE / site-based work. Start the workday, select the assigned site, start the site task, and close it with Complete, Stop, or Issue.
1

Clock In to Start the Day

After login, enable GPS and tap Clock In to start the DTE workday before selecting a site.

2

Select Planned Site

Select a site from My Planned Sites or Select Site to Start, then check GPS distance.

3

Start Site / Close Task

Tap Start Site to begin site work, then close with Complete, Stop, or Issue based on the field result.

4

Check History / Export

Review site records, time, status, and export CSV when evidence is required.

Per-Site Workflow

  • Log in and allow Location / Camera permissions.
  • Tap Clock In to start the workday.
  • Select the assigned site.
  • Confirm GPS is within the allowed radius.
  • Tap Start Site when actual work begins.
  • Close the task with Complete, Stop, or Issue.

Important Notes

  • You must Clock In for the day before starting a site.
  • Confirm the correct site before tapping Start Site.
  • If outside the allowed radius, the system may block site start.
  • Select the correct closing result: Complete, Stop, or Issue.
  • If you forget to close a site, contact your supervisor or Admin.
