ACE ClockApp คู่มือใช้งาน

Clock In / Clock Out PER-SITE

สำหรับ DTE / งานราย Site ใช้เปิดวันทำงาน เลือก Site ที่ได้รับมอบหมาย Start Site และปิดงานด้วย Complete, Stop หรือ Issue
1

Clock In เปิดวันทำงาน

หลัง Login ให้เปิด GPS แล้วกด Clock In เพื่อเริ่มวันของ DTE ก่อนเลือก Site

2

เลือก Planned Site

เลือก Site จาก My Planned Sites หรือ Select Site to Start และตรวจระยะ GPS

3

Start Site / ปิดงาน

กด Start Site เพื่อเริ่มงาน Site แล้วปิดด้วย Complete, Stop หรือ Issue ตามผลหน้างาน

4

ตรวจ History / Export

ตรวจประวัติราย Site, เวลา, สถานะ และ Export CSV เมื่อใช้เป็นหลักฐาน

ลำดับใช้งาน Per-Site

  • Login และอนุญาต Location / Camera
  • กด Clock In เพื่อเปิดวันทำงาน
  • เลือก Site ที่ได้รับมอบหมาย
  • ตรวจว่า GPS อยู่ในระยะที่กำหนด
  • กด Start Site เมื่อเริ่มงานจริง
  • ปิดงานด้วย Complete, Stop หรือ Issue

จุดที่ต้องระวัง

  • ต้อง Clock In เปิดวันก่อน จึงจะ Start Site ได้
  • เลือก Site ให้ถูกก่อนกด Start Site
  • ถ้าอยู่นอกรัศมี ระบบอาจไม่ให้เริ่ม Site
  • เลือกผลปิดงานให้ตรง: Complete, Stop หรือ Issue
  • ถ้าลืมปิด Site ให้แจ้งหัวหน้างานหรือ Admin
Clock Inเปิดวันทำงานของ DTE และยืนยันตัวตนก่อนเริ่ม Site แรก
Start Siteเริ่มจับเวลางานของ Site ที่เลือก สามารถมีหลาย Site ต่อวันได้
Complete / Stop / Issueใช้ปิดงานราย Site ตามผลจริง เพื่อให้รายงานและประวัติถูกต้อง

สแกนเพื่อเข้าใช้งาน ClockApp

เปิดผ่านมือถือเพื่อใช้งาน GPS, Camera และการตรวจระยะ Site ได้สมบูรณ์
