Leave Workflow

  • Log in to ClockApp and open the Leave tab.
  • Select the Leave Type that matches the reason.
  • Select the correct Start Date and End Date.
  • Enter a clear Reason so the approver can review quickly.
  • Check Total Days, then tap Submit Request.

After Submission

  • Check the request in My Leave Requests.
  • Use Refresh to update the latest status.
  • If the request is Pending, wait for the supervisor review.
  • Check Leave Statistics to see used leave days.
  • Read Leave Rules before submitting the next request.
