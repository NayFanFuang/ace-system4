Steps after HR sends a Welcome Email from HR Employees until the employee logs in and sets a new password for the first time.
1

HR Sends Welcome Email

Open HR Employees, select the employee, and click Send Welcome Email to send the initial username and password.

Set New Password

The system requires a password change before use. Enter Current, New, and Confirm Password.

For Employees

  • Open the ACE System Login Credentials email.
  • Use the Username and Password from the email for first login.
  • Set a new password with at least 8 characters, letters, and numbers.
  • Do not use the same password as email or employee code.
  • After saving, the system opens pages based on employee permission.
