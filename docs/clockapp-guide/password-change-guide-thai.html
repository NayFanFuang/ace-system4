สำหรับพนักงาน

  • เปิดอีเมลหัวข้อ ACE System Login Credentials
  • ใช้ Username และ Password จากอีเมลเพื่อ Login ครั้งแรก
  • ตั้งรหัสผ่านใหม่อย่างน้อย 8 ตัว มีตัวอักษรและตัวเลข
  • ห้ามใช้รหัสผ่านเหมือน Email หรือ Employee Code
  • หลังบันทึกสำเร็จ ระบบจะเข้าใช้งานได้ตามสิทธิ์ของพนักงาน

Flow โดยสรุป

HRเปิดหน้า HR Employees และเลือกพนักงาน
Emailระบบส่ง Username, Password และ Link เข้าใช้งาน
Loginพนักงาน Login ด้วยรหัสเริ่มต้นจากอีเมล
Changeตั้งรหัสใหม่ตามเงื่อนไขความปลอดภัย
Useเข้าใช้งาน ClockApp / HR / Project ตามสิทธิ์

สแกนเพื่อเข้าใช้งาน ClockApp
