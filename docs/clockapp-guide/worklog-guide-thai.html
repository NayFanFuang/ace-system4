Daily Work Log / Timesheet

คู่มือกรอกสรุปงานรายวันสำหรับพนักงาน DAILY ตั้งแต่เปิดหน้า Worklog ตรวจข้อมูล Clock กรอก Detail Work จนถึง Export Timesheet PDF
ใช้คู่กับ Clock DAILY ระบบจะใช้ข้อมูล Clock In/Out เพื่อปลดล็อกการกรอกงานและคำนวณชั่วโมงใน Timesheet

ขั้นตอนใช้งาน Worklog

เปิดจาก https://ace.airconnect-e.com/worklog/me

1

Login และเปิด Worklog

เข้าสู่ระบบ ACE ด้วยบัญชีบริษัท แล้วเข้าเมนู Daily Work Log หรือเปิดลิงก์ Worklog โดยตรง

2

เลือกเดือนและวันที่

เลือกเดือนที่ต้องการตรวจ ปฏิทินจะแสดงวัน Filled, Missing, Future clock และ No clock เพื่อให้ตามงานได้เร็ว

3

กรอก Detail Work

เลือกวันที่ Clock In/Out ครบ แล้วกรอกรายละเอียดงานแบบสั้น กระชับ ไม่เกิน 500 ตัวอักษร สามารถใส่ Project prefix ได้

4

Save และ Export PDF

กด Save หรือรอ Auto-save อัปโหลด My signature จากนั้นเลือก Period start และกด Export PDF เพื่อออก Timesheet

1
Clock ต้องครบก่อนถ้าวันนั้นยังไม่มี Clock Out หรือไม่มีข้อมูล Clock ช่อง Detail Work จะถูกล็อก และยัง Save ไม่ได้
2
Missing คือวันที่ต้องกรอกระบบนับวันที่มีข้อมูล Clock แต่ยังไม่มี Detail Work ให้กดวันที่สีแดงเพื่อกรอกย้อนหลัง
3
ลายเซ็นจำเป็นต่อ PDFต้องอัปโหลด My signature ก่อน ปุ่ม Export PDF จึงพร้อมใช้สำหรับออก Timesheet
4
เลือก Period start ให้ถูกเลือก 26 Payroll cycle สำหรับรอบเงินเดือน หรือ 1 Calendar month สำหรับรายเดือนปกติ

ควรกรอกอย่างไร

  • เขียนรายละเอียดงานจริงของวันนั้นแบบสั้นและตรวจสอบได้
  • ขึ้นต้นด้วย Project code เมื่อเกี่ยวข้องกับงานโครงการ
  • ตรวจวัน Missing ก่อนสิ้นรอบ เพื่อไม่ให้ Timesheet ขาดข้อมูล

ข้อควรระวัง

  • ถ้า Clock In/Out ไม่ครบ ให้แก้ไขเรื่อง Clock ก่อน
  • ห้ามกรอกวันอนาคต ระบบจะไม่อนุญาตให้บันทึก
  • Export PDF จะใช้ข้อมูลเดือนและ Period start ที่เลือกบนหน้า
